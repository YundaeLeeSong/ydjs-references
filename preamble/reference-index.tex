% ------------------------------------------------------------------------------
% Index setup (imakeidx)
% ------------------------------------------------------------------------------
% DESCRIPTION
%   Enables index generation with convenient printing and TOC integration.
%
% COMMON OPTIONS
%   title=Index   – custom index title (default: Index)
%   intoc         – include the index in the table of contents
%
% USAGE
%   1) Declare the index here via \makeindex[...]
%   2) Mark entries in your document using \index{term} or \index{term!subterm}
%   3) Print the index near the end using \printindex (see examples below)
%   Build notes: latexmk usually runs makeindex automatically. Manual build:
%     pdflatex file; makeindex file; pdflatex file
\makeindex[
  % title=Index,                % adjustable title (defualt: Index)
  intoc
]
